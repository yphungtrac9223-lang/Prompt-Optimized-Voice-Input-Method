## 二、桌面应用层

### Buzz（~17K ⭐）
- **GitHub**: https://github.com/chidiwilliams/buzz
- **技术栈**: Python, PySide6, 多 Whisper 后端
- **平台**: 全平台
- **定位**: 音视频文件转写工具，**不是**语音输入法（无热键+自动粘贴）
- **对比**: 同为 PySide6，但定位完全不同

### WhisperWriter（~1K ⭐）
- **GitHub**: https://github.com/savbell/whisper-writer
- **技术栈**: Python, 本地 Whisper
- **平台**: 全平台
- **定位**: 与本项目最接近——按快捷键说话+自动输入
- **对比**: 无 GUI、仅本地 Whisper（需显卡）、不支持云端 API、开发不活跃

### Whispering / Epicenter（~2K ⭐）
- **GitHub**: https://github.com/EpicenterHQ/epicenter
- **技术栈**: Svelte 5, Tauri (Rust), ~22MB 安装包
- **平台**: 全平台 + 浏览器扩展
- **对比**: 技术栈差异大（Tauri+Svelte），主要依赖本地 Whisper

### Handy（~1K ⭐）
- **GitHub**: https://github.com/cjpais/handy
- **技术栈**: Rust, Tauri, whisper.cpp
- **平台**: 全平台
- **定位**: 完全离线语音输入，按快捷键说话+自动粘贴
- **对比**: 强调纯离线，不支持云端 API

### FreeFlow（~1.5K ⭐）
- **GitHub**: https://github.com/zachlatta/freeflow
- **技术栈**: Swift (原生 macOS)
- **平台**: **仅 macOS**
- **对比**: 本项目覆盖 Windows，FreeFlow 完全缺失

### FluidVoice（~500 ⭐）
- **GitHub**: https://github.com/altic-dev/FluidVoice
- **技术栈**: Swift (原生 macOS)
- **平台**: **仅 macOS**
- **特色**: 支持 Nvidia Parakeet 模型，有 AI 文本润色（Command Mode）

### nerd-dictation（~2K ⭐）
- **GitHub**: https://github.com/ideasman42/nerd-dictation
- **技术栈**: Python, VOSK-API
- **平台**: **仅 Linux**
- **定位**: 极简离线语音听写，面向 Linux 终端用户

## 四、开源生态不足总结

1. **macOS 过度集中**: 高质量项目几乎全是 macOS 独占，Windows 选择极少
2. **过度依赖本地推理**: 需要 GPU 或高性能 CPU，轻薄本体验差
3. **缺乏云端 ASR 集成**: 极少项目支持调用云端 API
4. **中文场景几乎空白**: 所有项目以英文为主，无中文 ASR 引擎集成
5. **UI 体验参差不齐**: 很多无 GUI 或 GUI 简陋
6. **缺乏 AI 后处理**: 很少在转写后做文本润色、格式调整
