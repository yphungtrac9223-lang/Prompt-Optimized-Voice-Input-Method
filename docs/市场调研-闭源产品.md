# 闭源/商业语音输入工具市场调研

> 调研时间：2026-03-24

---

## 一、通用语音输入工具

### 1. 讯飞输入法（语音输入）

- **平台**：Windows、Mac、Android、iOS
- **核心功能**：实时语音转文字，支持普通话、英语、方言（粤语、四川话等）；语音悬浮球；中英混输；语音翻译
- **定价**：免费（含广告），会员去广告约 25 元/年
- **优点**：中文识别准确率业界领先；方言支持丰富；离线语音包可用
- **缺点**：桌面端体验远不如移动端；隐私争议大；不支持自定义词库/编程术语；标点符号智能插入不够准确
- **用户痛点**：广告多；PC 端语音功能是"二等公民"；专业术语识别差；无法在所有应用中使用

### 2. 搜狗输入法（语音输入）

- **平台**：Windows、Mac、Android、iOS
- **核心功能**：语音转文字，支持中英文；语音悬浮窗；同声传译功能
- **定价**：免费（含广告）
- **优点**：中文识别质量好；与搜狗生态整合；支持语音翻译
- **缺点**：PC 端语音功能不稳定；广告和弹窗严重；隐私问题；技术术语识别差
- **用户痛点**：弹窗广告太多；语音功能藏得深；英文识别远不如中文

### 3. Windows 语音识别 / Voice Typing（Win+H）

- **平台**：Windows 10/11
- **核心功能**：系统级语音输入（Win+H）；支持语音命令；Windows 11 改用云端识别提升质量
- **定价**：免费（系统自带）
- **优点**：零安装；Windows 11 版本识别质量有明显提升；支持自动标点
- **缺点**：中文支持质量一般；编程术语基本无法识别；语音命令仅支持英语；不支持自定义词汇
- **用户痛点**：中文识别太差；经常莫名其妙停止监听；不支持中英混输

### 4. macOS Dictation（听写功能）

- **平台**：macOS、iOS
- **核心功能**：系统级语音输入；支持离线识别（Apple Silicon）；支持语音命令
- **定价**：免费（系统自带）
- **优点**：Apple Silicon 上离线识别质量很好；系统级集成；隐私保护好
- **缺点**：中文识别准确率中等；不支持自定义词汇；中英混输体验差；无法自定义快捷键
- **用户痛点**：中英混输时经常出错；专业词汇识别不了

## 二、面向效率/专业场景的语音输入工具

### 5. SuperWhisper

- **平台**：macOS（仅 Apple Silicon）
- **核心功能**：基于 Whisper 模型的本地语音转文字；支持多种模型大小；全局快捷键触发；AI 文本后处理
- **定价**：Pro $8/月 或 $60/年；Lifetime $150
- **优点**：完全本地处理，隐私极好；多语言识别质量高；AI 后处理功能强大
- **缺点**：仅限 macOS + Apple Silicon；大模型占用内存高；没有实时流式输出；中文标点问题
- **用户痛点**：不支持 Windows；大模型太慢；偶尔出现 Whisper 幻觉；没有实时反馈

### 6. Wispr Flow

- **平台**：macOS（主要），Windows 版开发中
- **核心功能**：AI 驱动的语音输入；上下文感知——根据当前应用自动调整输出格式；支持语音命令
- **定价**：Pro $10/月 或 $96/年
- **优点**：上下文感知是核心卖点；识别质量很高；UI 精美
- **缺点**：依赖云端有延迟；隐私顾虑；订阅制价格不便宜；Windows 支持不成熟；编程场景支持有限
- **用户痛点**：太贵了；离线不能用；Windows 版太晚；对编程/代码场景几乎无优化

### 7. Talon Voice

- **平台**：Windows、macOS、Linux
- **核心功能**：面向程序员的语音+眼动控制编程工具；高度可编程/可定制（Python 脚本）；支持 Cursorless
- **定价**：免费（公测），Patreon $15/月
- **优点**：程序员语音编程的标杆产品；极度可定制；跨平台
- **缺点**：学习曲线极其陡峭；中文支持基本为零；社区驱动文档分散
- **用户痛点**：学习成本太高（入门需几周）；中文完全不行；配置太复杂

### 8. Nuance Dragon Professional

- **平台**：Windows（已停止零售，转向企业订阅）
- **核心功能**：老牌专业语音识别；支持自定义词汇和语音模型训练
- **定价**：$699 一次性（已停售）
- **优点**：专业领域词汇识别极强；支持自定义词汇训练
- **缺点**：被微软收购后消费级产品线基本放弃；价格极高；不支持 macOS；中文支持不佳
- **用户痛点**：太贵了；Mac 版被砍；技术停滞

## 三、总结

### 闭源产品普遍不足

| 痛点类别 | 具体表现 |
|---------|---------|
| **隐私问题** | 大多数需要将语音上传云端 |
| **中英混输差** | 几乎所有产品在中英文混合场景下表现糟糕 |
| **编程场景零优化** | 除 Talon 外，没有产品针对代码术语做专门优化 |
| **平台锁定** | SuperWhisper 仅 macOS，Dragon 仅 Windows |
| **定价不合理** | 订阅制 $8-15/月，Dragon $699 |
| **自定义词汇难** | 大多数不支持或很难添加自定义术语 |
| **标点智能化不足** | 中文标点自动插入普遍不够智能 |
| **无 AI 后处理** | 大多数只做"语音转文字"，不做文本优化 |

### 用户想要但没有的功能

1. 能识别编程术语、变量命名风格的语音输入
2. 中英混输高质量识别
3. 本地化 + 高质量并存
4. AI 智能后处理（适配上下文）
5. 跨平台一致体验
6. 轻松添加自定义词汇表
7. 与 AI 编程工具集成（Cursor、Claude Code 等）
8. 合理定价（一次性买断或低价/免费）
